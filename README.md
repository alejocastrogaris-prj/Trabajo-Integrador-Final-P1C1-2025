Este programa sirve para consultar datos de países que están guardados en un archivo CSV. Tiene un menú con las siguientes opciones:

1. Mostrar todos los países
2. Buscar un país por su nombre
3. Filtrar por continente
4. Filtrar según una cantidad de habitantes
5. Filtrar según una superficie determinada
6. Ordenar la lista de países
7. Ver estadísticas generales
8. Salir del programa

El usuario debe escribir el número de la opción que quiere usar y seguir las indicaciones en pantalla.

Ejemplo de e/s:
Seleccione una opción: 7

ESTADÍSTICAS:

- País con menor población: Chile (19,116,201 hab.)

   Europa: 1 países
   América del Norte: 1 países

La forma de uso es ingresando por el usuario el número de opcion que da el menú.
